Web-based, GPU-accelerated Digital Compositor. Vanilla JS and WebGL

Try it out here.

You can pan and zoom with the mouse. Right click on the checkerboard area to select a node to create. Images can be uploaded with the using the button at the top right ("Browse" in Firefox, "Choose File" in Chrome). Nodes can be connected by clicking and dragging from one pin to another, or by click on one pin, then selecting the other.

CHIP-8 Emulator. C++

CHIP-8 (Wikipedia) is a virtual machine originally developed to run on 8-bit microprocessors. Many popular games have been recreated for it, including Tetris and Pac-Man as shown.
